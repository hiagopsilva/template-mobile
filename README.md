> Este repositório é um template mobile com React Native, Typescript ( e umas libs adicionais ) para utilizar em projetos.

## Install

```sh
yarn
```

## Usage

*Android*
```sh
npx react-native run-android
```
*IOS*
```sh
npx react-native run-ios
```
